Corvex Store is a headless e-commerce platform that lets merchants build complete online stores through a visual drag-and-drop editor — and gives developers full control via LiquidJS templates, a component system called Sections, and a REST API. Whether you're a store owner composing pages without code, a theme developer building reusable components, or an integrator connecting external systems, this documentation covers everything you need.

## Get started in 4 steps

<Steps>
  <Step title="Understand the platform">
    Read [Core Concepts](/introduction/concepts) to learn how themes, sections, blocks, and the draft/publish model work together.
  </Step>
  <Step title="Set up your environment">
    Follow the [Quick Start](/introduction/quickstart) to spin up the editor and render backend locally, create a store, and add your first section.
  </Step>
  <Step title="Build your theme">
    Use the [Theme guide](/themes/overview) and [Sections reference](/sections/overview) to create reusable components with configurable settings.
  </Step>
  <Step title="Integrate via API">
    Use the [REST API](/api/overview) to manage stores programmatically, sync products, and listen to [webhooks](/api/webhooks) for real-time events.
  </Step>
</Steps>
