Corvex Store is a headless e-commerce platform that lets you build complete storefronts using a visual drag-and-drop editor, server-side LiquidJS templates, and a REST API. Whether you're a developer building reusable components or a store owner configuring pages without writing code, Corvex gives you the tools to ship quickly with full visual control.

## Who it's for

| Role | What you can do |
|---|---|
| **Theme developer** | Build custom sections in Liquid, configure layouts, define editor settings schemas |
| **Integrator** | Use the REST API to connect products, manage pages, and wire up custom domains |
| **Store owner** | Compose pages in the visual editor, publish with one click, manage content without code |

## What you can build

- Complete stores with a home page, product catalog, product detail pages, and content pages
- Reusable UI components (sections) with settings editable in the visual editor
- Themes exportable as a ZIP and importable into any store
- External integrations via REST API and real-time WebSocket events

<Note>
Corvex Store is optimized for **speed of creation** and **visual customization**. It is not a general-purpose headless commerce platform — it does not expose a GraphQL API or a storefront SDK, and it is not designed to replace high-volume platforms for complex catalog operations.
</Note>
